High-level functional requirements
Zapper functionality with a service list and basic service info
Support for both Broadcast and OTT services
Support for first-time service installation from all sources
Integrated into the Android Live TV App workflow
 

High-level technical requirements
Support for two TV Inputs – Broadcast and OTT
Each TV Input shall fully support TIF architecture
Broadcast TV Input utilizes Comedia DTV MW service for operation
OTT TV Input is a custom implementation working with MediaPlayer
Metadata shall be gathered and used to show basic service information (service name, description for OTT services)
App shall be demonstrated as an Android 16 app run from within Android Cuttlefish emulator
